# NeuroBreak - Frontend

##  Descripción
Frontend de la plataforma NeuroBreak desarrollado con React + Bootstrap.

##  Requisitos previos
- Node.js >= 16.x
- npm >= 8.x

##  Instalación
\\\ash
npm install
\\\

##  Ejecutar en desarrollo
\\\ash
npm start
\\\

##  Build para producción
\\\ash
npm run build
\\\

##  Estructura del proyecto
- **/src/config** - Configuraciones
- **/src/core** - Lógica de negocio y modelos
- **/src/infrastructure** - Servicios externos (API, Auth, Storage)
- **/src/presentation** - Componentes React y páginas
- **/src/state** - Gestión de estado (Context API)
- **/src/utils** - Utilidades generales

##  Tecnologías
- React 18
- Bootstrap 5
- React Router 6
- Axios
- Chart.js
